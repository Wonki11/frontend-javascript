공간 padding 여백 margin

        Padding(공간) - 내가 가진 범위 틀 안에서 여유있는 공간을 설정하는 것
            pt- : padding-top 태그 위쪽 패딩을 설정
            pr- : padding-right 태그 오른쪽 패딩을 설정
            pb- : padding-bottom 태그 아래쪽 패딩을 설정
            pl- : padding-left 태그 왼쪽 패딩을 설정

            px- : 태그의 왼쪽 오른쪽 패딩을 설정
            py- : 태그의 위'족과 아래쪽 패딩을 설정

             p- : 태그의 모든 방향 패딩을 설정
             
        Margin(여백) - 다른 태그나 다른 값과 거리를 두는 것
            mt- : padding-top 태그 위쪽 패딩을 설정
            mr- : padding-right 태그 오른쪽 패딩을 설정
            mb- : padding-bottom 태그 아래쪽 패딩을 설정
            ml- : padding-left 태그 왼쪽 패딩을 설정

            mx- : 태그의 왼쪽 오른쪽 패딩을 설정
            my- : 태그의 위'족과 아래쪽 패딩을 설정

            m- : 태그의 모든 방향 패딩을 설정
